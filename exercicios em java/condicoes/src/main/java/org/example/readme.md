# Estruturas de Controle
* Condicional:

 ```
if (condicao) {
    // bloco de código
} else if (outraCondicao) {
    // outro bloco
} else {
    // bloco padrão
}
```

* Switch:
```
switch (variavel) {
    case valor1:
        // código
        break;
    case valor2:
        // código
        break;
    default:
        // código padrão
}
```

* Laços:

 * for:
 ```

 for (int i = 0; i < 5; i++) {
    // código
 }
```

* while:
```
while (condicao) {
    // código
}
```

* do-while:
```
do {
    // código
} while (condicao);
```
